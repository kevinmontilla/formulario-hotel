# Formulario de Reservación de Hotel

Este proyecto consiste en la maquetación de una interfaz web para realizar reservaciones de hotel. Es una aplicación estática centrada en el diseño visual y la experiencia de usuario (UI/UX) mediante el uso de HTML y CSS estándar.

> **Nota:** Este es un proyecto frontend con fines educativos. El formulario no procesa datos en el servidor (backend).

## Características

- **Diseño Visual:** Interfaz limpia y estructurada para la captura de datos de reservación.
- **Campos Completos:** Incluye selección de fechas, tipos de habitación, extras.
- **Estilos Personalizados:** Uso de `estilo.css` para modificar la apariencia.
- **Tipografía:** Integración de fuentes personalizadas (carpeta `fuentes`).
- **Recursos Gráficos:** Gestión de imágenes optimizadas para la web, incluyendo un carrusel y logo creado desde 0.

## Tecnologías Utilizadas

* ![HTML5](https://img.shields.io/badge/html5-%23E34F26.svg?style=for-the-badge&logo=html5&logoColor=white) **HTML5:** Estructura semántica del contenido.
* ![CSS3](https://img.shields.io/badge/css3-%231572B6.svg?style=for-the-badge&logo=css3&logoColor=white) **CSS3:** Hojas de estilo para el diseño visual y layout.

## Estructura del Proyecto

```text
formulario-hotel/
├── fuentes/         # Tipografías locales utilizadas
├── img/             # Imágenes y recursos gráficos
├── main/            # Página principal, donde se rellena el formulario
├── reservacion/     # Página de "confirmación" de reserva
├── index.html       # Página inicial donde se "inicia sesión"
├── estilo.css       # Hoja de estilos de la página inicial
└── README.md        # Documentación del proyecto
```
